## English Overview
This repository contains a comprehensive cheatsheet for Numerical Methods and Scientific Computing, designed in LaTeX. It is a dense, unified reference covering everything from fundamental computer arithmetic to advanced techniques for solving Ordinary and Partial Differential Equations.

**Main Topics:**
* Scientific Computing and Computer Arithmetic (IEEE 754, Error Analysis).
* Root Finding and Optimization (Newton-Raphson, Secant, Brent's Method).
* Matrix Algebra and Linear Systems (LU, SVD, Krylov Subspaces).
* Numerical Differentiation, Integration (Quadrature), and Interpolation.
* Ordinary Differential Equations (RK4, Stiff Problems) and Partial Differential Equations (Finite Differences).

## Resumen en Español
Este repositorio contiene un formulario exhaustivo de Métodos Numéricos y Cálculo Científico diseñado en LaTeX. Es una referencia unificada que abarca desde la aritmética computacional fundamental hasta técnicas avanzadas para la resolución de Ecuaciones Diferenciales Ordinarias y Parciales.

**Contenido Principal:**
* Cálculo Científico y Aritmética Computacional (IEEE 754, Análisis de Error).
* Búsqueda de Raíces y Optimización (Newton-Raphson, Secante, Método de Brent).
* Álgebra Matricial y Sistemas Lineales (LU, SVD, Subespacios de Krylov).
* Diferenciación Numérica, Integración (Cuadratura) e Interpolación.
* Ecuaciones Diferenciales Ordinarias (RK4, Problemas Rígidos) y Ecuaciones Diferenciales Parciales (Diferencias Finitas).
